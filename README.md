# Firefly
---
## Installation

```
git clone https://github.com/Jupiter79/Firefly
cd Firefly
npm install
```

---
## Start bot

```
npm run start
```

---
## Slash Commands

### Update Commands
```
npm run deploy_commands
```

### Reset/Delete Commands
```
npm run delete_all_commands
```
