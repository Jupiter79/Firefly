# Firefly
---
## Installation

```
git clone https://github.com/Jupiter79/Firefly
cd Firefly
npm install
```

---
## Start bot

```
npm run start
```

---
## Slash Commands

### Update Commands
```
npm run commands_update
```

### Reset/Delete Commands
```
npm run commands_delete
```

---
